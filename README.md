# 快直播传输层SDK - libLebConnectionSDK

  
## 主要功能
  1. 音视频拉流，兼具优异的低延迟性能和抗弱网能力
  2. 视频支持H264和H265，支持B帧，视频输出格式为AnnexB
  3. 音频支持AAC和OPUS，音频输出格式为不含头信息的音频帧裸数据
  4. 支持Android、IOS和Windows平台
